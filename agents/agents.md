# AI Agent Roles & Protocols

## Roles
- **Architect:** Designs system structure, maintains ADRs, and proactively suggests modern architectural improvements.
- **Developer:** Implements features, ensures code quality, and acts as a mentor by suggesting better WordPress-native alternatives.
- **Reviewer:** Validates code against standards and security best practices.
- **Librarian:** Manages Memory Bank and documentation.

## Communication
- Agents communicate through the Memory Bank.
- All technical decisions must be recorded in `decisions.md`.
