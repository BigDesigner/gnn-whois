# GNN Whois

A premium, minimalist WordPress plugin to perform domain WHOIS lookups directly from your website. Built with performance, security, and universal theme compatibility in mind.

## Features
- **Modern Pill Search:** Elegant, pill-shaped search bar with floating focus animations and interactive labels.
- **Natural Result Flow:** WHOIS data flows seamlessly within your theme's content area without restrictive frames.
- **Pure CSS Icons:** Lightweight, high-performance icons built entirely with CSS.
- **Universal Adaptive UI:** Modern design that automatically adapts to your theme's typography and color scheme (Light/Dark mode).
- **Multi-TLD Support:** Native support for .com, .net, .org, .tr, .io, .ai, .dev, .app, and many more.
- **Performance:** Integrated Transient Caching (1 hour) for optimized lookup speed.
- **Security:** Sanitized inputs, escaped outputs, and nonce-protected admin actions.
- **GitHub Updates:** Built-in updater for seamless version management via GitHub Releases.

## Installation
1. Upload the `gnn-whois` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

## Usage
Simply use the shortcode `[gnn_whois]` anywhere in your posts, pages, or widgets.
