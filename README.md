# GNN Whois

A premium, minimalist WordPress plugin to perform domain WHOIS lookups directly from your website. Built with performance, security, and universal theme compatibility in mind.

## Features
- **Domain WHOIS Lookup:** Retrieve detailed registration information for domains across various TLDs.
- **Shortcode Powered:** Easily embed the lookup form anywhere using the `[gnn_whois]` shortcode.
- **Multi-TLD Support:** Native support for .com, .net, .org, .tr, .io, .ai, .dev, .app, and many more.
- **Premium Adaptive UI:** Modern "Glassmorphism" design that automatically adapts to your theme's light or dark mode.
- **Copy to Clipboard:** One-click functionality to copy WHOIS results with interactive feedback.
- **Performance:** Integrated Transient Caching (1 hour) to reduce network load and improve speed.
- **Security:** Sanitized input handling, nonces for admin actions, and properly escaped output.
- **GitHub Updates:** Built-in updater for easy management and manual update checks via GitHub Releases.

## Installation
1. Upload the `gnn-whois` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

## Usage
Simply use the shortcode `[gnn_whois]` anywhere in your posts, pages, or widgets.
